# Invitacion-boda-Teresa-y-Alberto

Invitacion web estatica para la boda de Teresa y Alberto, desarrollada con HTML, CSS y JavaScript.

## Estructura del proyecto

- `index.html`: pagina principal.
- `css/estilo.css`: estilos del proyecto.
- `js/js.js`: logica e interacciones.
- `recursos/`: imagenes, textura y elementos graficos.

## Recursos externos en uso

### Google Fonts

Se utilizan las siguientes fuentes cargadas desde Google Fonts:

- `Playfair Display`
- `Raleway`
- `Parisienne`

Referencia:

- [Google Fonts](https://fonts.google.com/)

Uso:

- Las fuentes se consumen desde CDN y no se incluyen localmente en el repositorio.
- Sus licencias deben revisarse en la ficha oficial de cada familia si en algun momento se redistribuyen los archivos de fuente.

### Google Maps

El proyecto incluye enlaces externos a Google Maps para la ubicacion de los eventos.

Uso:

- No se incrustan mapas dentro del sitio.
- Solo se redirige al usuario mediante enlaces externos.
